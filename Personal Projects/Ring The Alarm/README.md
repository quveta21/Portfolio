## 📝 Project Overview
In Intensive Care Units (ICUs), medical staff are overwhelmed daily by hundreds of alarms from independent systems (cardiac monitors, ventilators, lab systems). Many of these alerts describe the exact same clinical event using different wording or sources, leading to severe **alarm fatigue**.

**Our Solution:** An intelligent system for the **semantic deduplication of medical alarms** in ICUs, powered by Language Models (LLMs) and embeddings. The system processes, clusters, and deduplicates redundant alerts, delivering them through an intuitive web interface to improve ICU workflows.
